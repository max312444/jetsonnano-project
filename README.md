🚗 Jetson Nano 자율주행 자동차 프로젝트
이 저장소는 Jetson Nano를 활용한 자율주행 RC 자동차 프로젝트에 관련된 모든 코드, 데이터 및 문서를 포함합니다.
카메라로 감지한 라인을 추적하고, CNN 기반의 모델을 학습시켜 조향 및 속도 제어를 자동화하는 것을 목표로 하고 있습니다.
