# 🚗 Jetson Nano 자율주행 자동차 프로젝트

이 저장소는 Jetson Nano를 활용한 자율주행 RC 자동차 프로젝트와 관련된 모든 코드, 데이터, 그리고 문서화를 포함합니다.  
라인 트레킹, 조향 및 속도 제어를 CNN 기반으로 학습하여 구현하는 것을 목표로 하고 있습니다.
