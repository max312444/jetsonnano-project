🚗 Jetson Nano 自律走行車プロジェクト
このリポジトリは、Jetson Nano を利用した自律走行 RC カープロジェクトに関するすべてのコード、データ、ドキュメントを含みます。
カメラで取得したラインを追跡し、CNN ベースのモデルを学習させることで、ステアリングと速度制御を自動化することを目指しています。
